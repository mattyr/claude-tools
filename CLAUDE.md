# Marginalia

Inline review comment tool for markdown files.

## When you see a `<!-- REVIEW COMMENTS ... END REVIEW COMMENTS -->` block

This block contains numbered comments referencing specific quoted passages.
Each comment has a quoted string (the anchor) and feedback after the `→` arrow.

To address comments:
1. Find each quoted passage in the document
2. Revise it according to the feedback
3. If feedback is a question, use your judgment
4. Remove the comment block from the final output
